# react-smart-window
A Desktop-like draggable container for React

```js
import Window,{Draggable} from 'react-smart-window'
// create a draggable window.
<Window Title='Title'>
  ...React.Node
</Window>

// makes node draggable
<Draggable>
  ...React.Node
</Draggable>
```
### Compatibility
|Version     | Compatibility|
|------------|--------------|
|0.x         |     React    |

### Installing
```bash
$ npm install react-smart-window
```
